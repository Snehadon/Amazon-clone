# Amazon-clone
Builded an Amazon clone using HTML and CSS, replicating the visual structure of the original site.
Created a header with logo and navigation, product listings with details, and a footer.
Applied Amazon's color scheme and styling for a similar appearance. Included a search bar, and ensure responsiveness for different devices.
Added hover effects to buttons and links for interactivity. Focus on front-end development, excluding actual e-commerce functionality.
